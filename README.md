# 🚀 AutoDS Agent - 快速开始指南

## 前端优化说明

本次更新将 Agent 控制台重构为**对话式交互界面**，主要改进：

### ✨ 核心特性

1. **对话式交互**
   - 所有交互（目标输入、配置、HITL响应）统一在对话流中完成
   - 类似 ChatGPT 的自然交互体验

2. **智能消息折叠**
   - 主要步骤（Planning、Execution等）以卡片形式展示
   - 子步骤（search、agent selection等）自动归类并可折叠
   - 点击步骤卡片展开/收起详情

3. **顶部进度条**
   - 5阶段可视化：初始化 → 研究 → 规划 → 执行 → 完成
   - 实时状态更新（pending/active/completed/error）
   - 彩色连接线指示进度流转

4. **侧边文件管理**
   - 文件浏览器从侧边滑出，不遮挡主界面
   - 保留所有文件操作功能（上传/预览/下载/删除）

5. **内置配置面板**
   - 点击设置按钮弹出配置面板
   - 无需切换页面即可调整参数

## 启动步骤

### 1. 安装依赖（首次运行）

```bash
cd frontend
npm install
```

### 2. 启动后端服务

```bash
cd ../src
python server.py
```

后端运行在 `http://localhost:8000`

### 3. 启动前端

```bash
cd ../frontend
npm run dev
```

前端运行在 `http://localhost:3000`

### 4. 或使用一键启动脚本

```bash
# 在项目根目录
./start.sh
```

## 使用流程

### 初次使用

1. 打开 `http://localhost:3000`
2. 看到欢迎界面，展示4个示例任务
3. 点击示例或直接输入您的任务
4. 按 Enter 或点击发送按钮

### 配置调整

1. 点击输入框左侧的 ⚙️ 按钮
2. 勾选/取消需要的功能
3. 配置立即生效

### 查看执行

1. **顶部进度条** - 显示当前在哪个阶段
2. **对话消息** - 详细执行日志
3. **可折叠步骤** - 点击主步骤查看子步骤详情

### HITL 交互

当看到黄色高亮的提示消息时：
1. 输入框会显示提示信息
2. 直接输入您的响应（如 "approve"）
3. 按 Enter 继续

### 查看文件

1. 点击右上角 📁 按钮
2. 侧边栏滑出显示文件列表
3. 可预览/下载/删除文件
